# Choix d'extensions

Ce document décrit comment mettre en place un environnement de travail LaTeX fonctionnel basé sur la distribution TEX Live, l’éditeur TeXstudio et les fichiers de style preambule_college.sty et
preambule_personnalisation.sty .

Il regroupe des trucs et astuces qui m’ont pris du temps à mettre au point.
